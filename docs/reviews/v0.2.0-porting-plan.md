```r
#' Default DBH class breaks (cm) for 10-class SSD
#' @export
SG_DBH_BREAKS <- c(0, 8, 12, 16, 20, 24, 28, 32, 40, 52, Inf)

#' Default DBH class midpoints (cm) for 10-class SSD
#' @export

#' Default lower threshold DBH (cm) for bimodal SSD classifier
#' @export
SG_DBH_THETA <- 8L
```

ADR-002 note: these are pure R values, no keras dependency.
